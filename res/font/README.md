# Font Generation for _socc on croc_

Just run the script, e.g.

```sh
python ttf_to_svrom.py --font-path CLEagleIII/Px437_CL_EagleIII_8x16.ttf --output-sv ./out/CLEagleIII_8x16_rom.sv
```
